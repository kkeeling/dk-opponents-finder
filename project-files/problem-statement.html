DraftKings Opponent Finder Chrome Extension: Problem Statement

Background

DraftKings is a popular platform for daily fantasy sports contests. However, it can be challenging for users to quickly identify contests with less experienced opponents, which may offer better chances of winning.

Problem

Currently, users have to manually check each contest to determine the experience level of opponents. This process is time-consuming and inefficient, potentially causing users to miss out on favorable contests.

Proposed Solution

Develop a Chrome extension that overlays opponent experience data directly on the DraftKings lobby. This extension will:

  • Display the number of opponents with different experience levels (represented by badges) for contests with 5 or fewer entries.
  • Provide visual cues to quickly identify contests with less experienced opponents.
  • Integrate seamlessly with the DraftKings website, enhancing the user experience without disrupting the platform's functionality.

Benefits

  • Save time for users by eliminating the need to manually check each contest.
  • Improve decision-making by providing quick access to opponent experience information.
  • Enhance user experience on the DraftKings platform.

Constraints

  • Must comply with DraftKings' terms of service and API usage policies.
  • Should not interfere with the normal functioning of the DraftKings website.
  • Must handle authentication with DraftKings if required.
